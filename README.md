# Ogt

Open Graph Tags

## Installation

Add this line to your application's Gemfile:

    gem 'ogt'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ogt

## Usage

Add the og prefix in your layout
  
  <html prefix="og: http://ogp.me/ns#">
